### Standard usage 

To test, use the booking handler debug.

A booking location is the only strict requirement to display the form. If an appoinmtent type is not set, the generic "reasonForVisit" name will be used when fetching the form schema from the location 

```jsx harmony
import { DemoStandard } from "./readme";
<DemoStandard />;
```
